O que é o Box-e? é um serviço de armazenamento de notas fiscais eletrônicas para os clientes do escritório contábil, totalmente integrado com o Módulo Escrita Fiscal.

  • Como habilitar as empresas para utilizar o BOX-e?
  • ENVIO DE NF-e
    • Após habitar a empresa, seu cliente deverá enviar as notas para o BOX-e. Para isso, ele terá as alternativas:
    • Primeira opção é enviar um e-mail com as notas para o e-mail criado para o seu BOX-e (item 4 desta solução)
    • Segunda opção é realizada pela utilização do aplicativo ENVIA BOX-E.
    • Terceira opção é utilizar o box-e em sincronismo com a ferramenta do Busca NF-e

O que é o Busca NF-e? Todas as Notas Fiscais Eletrônicas emitidas em nome do seu cliente são transferidas automaticamente da Receita Federal para o seu escritório, ficando disponíveis para serem importadas no sistema, sem qualquer intervenção manual.

O que é o Onbalance? É o software em nuvem feito para seu cliente organizar e controlar a gestão financeira do negócio dele de uma forma mais simples, trazendo benefícios e menos preocupações para você.


O que é o Centro de Treinamento? É um serviço que trará praticidade para treinar seus colaboradores: treinamento online sobre os módulos do Contábil com certificação e tutoria.
